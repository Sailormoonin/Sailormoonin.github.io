#Python学习 爬虫初识

##郑重声明：身为一个鶸，还没有学完python，笔记仅为初步了解

##什么是爬虫


首先在百度百科上我们可以查到这么一句话
“网络爬虫是一段程序，它模仿人类访问互联网的形式，不停的从网上抓取想要的数据，我们可以定制各种各样的爬虫，来满足不同的需求，前提是法律允许”
简而言之，这个网络爬虫就是一种可以利用python来和网站进行交互的一种程序,爬虫的手速是非常快的，当我们发出想要的声音的时候，爬虫就能够把网页中我们想要的制定文本和图片进行保存

##网络爬虫都做了什么！
像我们经常在网上搜索一些资源的人都会知道，利用迅雷，磁力链接能够让我们很快的搜索到网络当中我们想要的资源
再比如我们也可以利用网络爬虫来采集一些对我们有用的信息

##在爬虫之前

###浏览网页的过程
在用户浏览网页时，我们会看到一些奇奇怪怪的东西，在我们发出想要的声音的时候，输入了一个网址，经过DNS服务器的域名解析（域名解析协议为IP地址）。
三次握手
穿过MAC的ARP协议成为链路层的一缕mac地址，穿行于各个网关之间，最终到达目的主机，而主机收到这个想要的声音之后，利用IP->TCP。TCP协议单元回应请求应答报文，在标志SYN,ACK都是1的时候，表示连接成功，于是这个想要的声音的回答从原路返回，而本主机收到这个回答之后，以IP->TCP，TCP协议单元回应请求确认的报文，而这个与目的主机相互交互的过程就是三次握手
如图所示

浏览器首先向域名发出HTTP请求，这个请求到达主机之后，主机用相同的方式回应，也就是从请求信息中获得客户机想访问的主机名。从请求信息中获取客户机想要访问的web应用（web应用程序指提供浏览器访问的程序，简称web应用）。从请求信息中获取客户机要访问的web资源。（web资源，即各种文件，图片，视频，文本等）读取相应的主机下的web应用，web资源。用读取到的web资源数据，创建一个HTTP响应。然后这个HTML数据通过TCPIP以及一系列活动到达本主机，本主机收到数据帧，通过IP tcp http 浏览器就可以用网页的形式显示HTML的内容，也就是说，我们所看到的网页实际上是用HTML代码写成的，网络爬虫爬的就是这些内容，网络爬虫通过分析，过滤这些代码，来获取图片，文字资源

###什么是URL
URL（Uniform Resource Locator）又称为唯一资源定位符，也就是我们所说的网址，但是域名和地址有一定的区别，域名相当于是符号化的IP地址，方便人们的记忆，但是URL地址指出文件的位置以及浏览器应该怎么处理它
URL的格式由三部分组成：
①第一部分是协议(或称为服务方式)。
②第二部分是存有该资源的主机IP地址(有时也包括端口号)。
③第三部分是主机资源的具体地址，如目录和文件名等。
